# irvanudin-buku_telepon

### Software yang dibutuhkan

- [Python v3.7.0](https://www.python.org/downloads/release/python-370)
- [Poetry (versi terbaru)](https://python-poetry.org)

### Inisialisasi

```
poetry install
```

### Run Server

```
poetry run python manage.py runserver
```
